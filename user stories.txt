User stories



* As a Rider or Driver, I should be able to add, edit, delete my SCHEDULE

* As a Rider or Driver, I should be able to view, edit, delete my ACCOUNT
* As a Rider or Driver, I should be able to change my PASSWORD



Rider Profile:
- Name
- Email
- Phone
- Address




* As a Rider, I should be able to SEARCH a driver near a set location
* As a Driver, I should be able to SEARCH a rider near a set location
* As a Rider, I should be able to see the driver's profile and schedule
* As a Driver, I should be able to see the rider's profile and schedule

* As a Rider or Driver, I should be able to contact (or message) the rider/driver
